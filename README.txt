HOW TO USE THIS PORTFOLIO SITE

1. Upload all files in this folder to your GitHub repository named:
   yourusername.github.io

2. Keep these files together at the top level:
   - index.html
   - style.css
   - script.js
   - Faith_Olawoyin_CV.pdf

3. Put your demo videos inside the assets folder with these names:
   - assets/pressure-app-demo.mp4
   - assets/telemanipulated-robot-demo.mp4

4. Optional poster images:
   - assets/pressure-app-poster.jpg
   - assets/robot-poster.jpg

5. In GitHub:
   - Go to Settings > Pages
   - Set Source to Deploy from a branch
   - Choose main branch and /(root)

6. Your site will appear at:
   https://yourusername.github.io
