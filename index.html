Selected Work

Projects that combine physical systems, electronics, firmware, and analysis

Flagship Project

Fiber-Optic Lock-In Amplifier Pressure Sensor & Calibration System

Developed an end-to-end diaphragm-based fiber-optic pressure sensing platform for 0–4000 psi measurements. The system integrates optical transduction, analog signal conditioning, embedded IQ demodulation on STM32, synchronized DAQ calibration, and nonlinear calibration modeling.

  • 30 kHz carrier generation and coherent demodulation
  • Cascaded IIR filtering and PLL-based signal recovery
  • Synchronized calibration workflow linking UART data with DAQ pressure acquisition
  • High-quality cubic calibration fit with strong full-range performance
STM32 Signal Processing DAQ Calibration
MAG: 70.0 mV Pressure: 0 psi
Software + Instrumentation

Live Pressure Monitoring Desktop Application

Built a modular Windows desktop application for live pressure monitoring from serial sensor data. The interface supports smoothing, cubic calibration, automatic microcontroller detection, and engineering unit conversion across psi, Pa, and bar for practical real-time operation.

Python Desktop UI Serial
Modeling

Finite-Strain Compression Modeling of Biomass Dewatering

Developed large-deformation models for biomass compression and dewatering using coupled nonlinear equations that capture solid deformation, pore-fluid behavior, and moisture extraction under applied load. The work targets physically informed prediction of dewatering efficiency and improved compression protocols.

MATLAB PDEs Poromechanics

Barrel-Bar Pressure and Temperature Sensor

Designed and characterized a combined pressure-temperature sensing assembly using a polished diaphragm, bonded strain gauge, co-located RTD, signal conditioning circuitry, and hydraulic chamber calibration for dual-parameter measurement.
